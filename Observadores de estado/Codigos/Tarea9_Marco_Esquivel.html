Control por realimentación del estado¶

Se considera el sistema cuya funcin de transferencia es $$ \begin{split} \frac{Y(s)}{U(s)}&=\frac{2}{(s+0.5)(s+0.1)(s+5)}\\ &=\frac{2}{s^3+5.6s^2+3.05s+0.25} \end{split} $$ Y cuya respresentacin en variables de estado es $$ \dot{\textbf{x}} = A\textbf{x} + Bu(t)\\ y = C\textbf{x} $$ Donde $$A = \left[\begin{matrix}0 & 1 & 0\\0 & 0 & 1\\-0.25 & -3.05 & -5.6 \end{matrix}\right]$$ $$B=\left[\begin{matrix}0\\0\\1 \end{matrix}\right]$$ $$C=\left[\begin{matrix}2 & 0 & 0 \end{matrix}\right]$$

Estabilidad del sistema¶

Para que el sistema pueda ser estabilizable mediante la realimentacin por variables de estado, el sistema debe ser controlable y observable. La matriz de controlabilidad del sistema es $$ \mathcal{C}= \left[\begin{matrix} 0 & 0 & 1\\ 0 & 1 & -5.6\\ 1 & -5.6 & 28.31 \end{matrix}\right] $$

La matriz de observabiliad del sistema es $$ \mathcal{O}= \left[\begin{matrix} 2 & 0 & 0\\ 0 & 2 & 0\\ 0 & 0 & 2 \end{matrix}\right] $$

Ya que el determinante de ambas matrices es diferente de cero, se dice que el sistema es controlable y observable.

Observador en lazo abierto con $u(t)=0$¶

Simulacin en variales de estado considerando $u(t)=0$ y $$ \textbf{x}(0)=\left[\begin{matrix}x_1(0)\\x_2(0)\\x_3(0)\end{matrix}\right] =\left[\begin{matrix}1\\0.5\\-1\end{matrix}\right] $$

El observador en lazo abierto genera un vector de estados estimados a partir de una estimacin inicial del estado y de la siguiente expresin. $$ \dot{\hat{\textbf{x}}}=A\hat{\textbf{x}} + Bu $$

Esta duplicacin resulta en un diagrama de bloques como la que se muestra a continuacin.

Observador en lazo cerrado¶

Se usa el valor $y(t)$ para mejorar el diseo del observador introduciendo una correccin proporcional al error de estimacin de la salida. El sistema resultante se muestra en el siguiente diagrama de bloques.

El valor estimado del vector de estados es entonces $$ \dot{\hat{\textbf{x}}}(t) = A\hat{\textbf{x}}(t) + Bu(t) + L[y(t) - C\hat{\textbf{x}}(t)] $$

def x_hat_byEuler(x0,x0_hat,t):
    x = np.zeros((3, len(t)))
    y = np.zeros(len(t))
    
    x_hat = np.zeros((3, len(t)))
    y_hat = np.zeros(len(t))
    
    h = t[1] - t[0]
    
    xn = x0
    yn = 2*xn[0]
    x[0,0] = xn[0]
    x[1,0] = xn[1]
    x[2,0] = xn[2]
    y[0] = yn
    
    xn_hat = x0_hat
    yn_hat = 2*xn_hat[0]
    x_hat[0,0] = xn_hat[0]
    x_hat[1,0] = xn_hat[1]
    x_hat[2,0] = xn_hat[2]
    y_hat[0] = yn_hat
    
    for n in range(1, len(t)):        
        ## Vector de estados real
        xn1 = xn + h*(A.dot(xn) + B*u)
        x[0,n] = xn1[0]
        x[1,n] = xn1[1]
        x[2,n] = xn1[2]
        
        yn = 2*xn1[0]
        y[n] = yn
        
        xn = xn1
        
        # Vector de estados estimado
        xn1_hat = xn_hat + h*(A.dot(xn_hat) + B*u + L*(yn - yn_hat))
        x_hat[0,n] = xn1_hat[0]
        x_hat[1,n] = xn1_hat[1]
        x_hat[2,n] = xn1_hat[2]
        
        yn_hat = 2*xn1_hat[0]
        y_hat[n] = yn_hat
        
        xn_hat = xn1_hat
    
    return x, y, x_hat, y_hat

def x_byEuler(x0, x0_hat, r, t):
    x = np.zeros((3, len(t)))
    y = np.zeros(len(t))
    
    x_hat = np.zeros((3, len(t)))
    y_hat = np.zeros(len(t))
    
    h = t[1] - t[0]
    
    xn = x0
    yn = 2*xn[0]
    x[0,0] = xn[0]
    x[1,0] = xn[1]
    x[2,0] = xn[2]
    y[0] = yn
    
    xn_hat = x0_hat
    yn_hat = 2*xn_hat[0]
    x_hat[0,0] = xn_hat[0]
    x_hat[1,0] = xn_hat[1]
    x_hat[2,0] = xn_hat[2]
    y_hat[0] = yn_hat
    
    chi_n = 0.
    
    KI = 90.
    for n in range(1, len(t)):
        # Vector de estados real
        u = chi_n*KI - K.dot(xn)
        
        xn1 = xn + h*(A.dot(xn) + B*u)
        x[0,n] = xn1[0]
        x[1,n] = xn1[1]
        x[2,n] = xn1[2]
        
        yn = 2*xn1[0]
        y[n] = yn
        
        xn = xn1
        
        # Vector de estados estimado
        u_hat = chi_n*KI - K.dot(xn_hat)
        
        xn1_hat = xn_hat + h*(A.dot(xn_hat) + B*u_hat + L*(yn - yn_hat))
        x_hat[0,n] = xn1_hat[0]
        x_hat[1,n] = xn1_hat[1]
        x_hat[2,n] = xn1_hat[2]
        
        yn_hat = 2*xn1_hat[0]
        y_hat[n] = yn_hat
        
        xn_hat = xn1_hat
        
        chi_n = chi_n + h*(r - yn)
    
    return x, y, x_hat, y_hat

Esquema Controlador-Observador¶

Se desea aplicar el esquema controlador-observador de acuerdo al siguiente diagrama, para una entrada de referencia tipo escalon unitario. Los valores en las ganancias del controlador se conservarn de acuerdo a las ya establecidas mientras que se aplicar una comparativa entre un observador lento y uno rpido.

A_BK = A - B.dot(K)
N = -1./(C.dot(np.linalg.inv(A_BK)).dot(B))

def con_obs(x0, x0_hat, r, t):
    x = np.zeros((3, len(t)))
    y = np.zeros(len(t))
    
    x_hat = np.zeros((3, len(t)))
    y_hat = np.zeros(len(t))
    
    h = t[1] - t[0]
    
    xn = x0
    yn = 2*xn[0]
    x[0,0] = xn[0]
    x[1,0] = xn[1]
    x[2,0] = xn[2]
    y[0] = yn
    
    xn_hat = x0_hat
    yn_hat = 2*xn_hat[0]
    x_hat[0,0] = xn_hat[0]
    x_hat[1,0] = xn_hat[1]
    x_hat[2,0] = xn_hat[2]
    y_hat[0] = yn_hat
    
    for n in range(1, len(t)):
        u = r*N - K.dot(xn_hat)
        
        # Vector de estados real        
        xn1 = xn + h*(A.dot(xn) + B*u)
        x[0,n] = xn1[0]
        x[1,n] = xn1[1]
        x[2,n] = xn1[2]
        
        yn = 2*xn1[0]
        y[n] = yn
        
        xn = xn1
        
        # Vector de estados estimado        
        xn1_hat = xn_hat + h*(A.dot(xn_hat) + B*u + L*(yn - yn_hat))
        x_hat[0,n] = xn1_hat[0]
        x_hat[1,n] = xn1_hat[1]
        x_hat[2,n] = xn1_hat[2]
        
        yn_hat = 2*xn1_hat[0]
        y_hat[n] = yn_hat
        
        xn_hat = xn1_hat
    
    return x, y, x_hat, y_hat
