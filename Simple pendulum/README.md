# Simple Damped Pendulum

Run *SimplePendulum.m* to play simulation.<br /><br />
\begin{equation}
x+1 = 2 \\
y+2 = 3 
\end{equation}
Default parameters:<br />
$g = 9.8$
\\[l = 1\\]
\\[$<br />$\\]
\\[m = 1\\]
\\[b = 0.25\\]
<br /><br />

These parameters can be changed in lines 2-5 in *SimplePendulum.m*
